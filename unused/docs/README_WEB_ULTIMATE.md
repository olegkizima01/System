## ⚡ Швидкий старт

### 1. Запустити веб-інтерфейс
```bash
cd /Users/dev/Documents/GitHub/System
./launch.sh
```
Обрати **[1] Web Interface**

### 2. Відкрити в браузері
```
http://localhost:8888
```

### 3. Натиснути MEGA BUTTON
- 🌊 **WINDSURF** - для Windsurf
- 💻 **VS CODE** - для VS Code

### 4. Дочекатися завершення
⏳ 5-10 хвилин

### 5. Перезавантажити Mac
```bash
sudo reboot
```

### 6. Встановити IDE заново
✅ Готово! Ви новий користувач!

## ⚠️ Важливо

### Перед запуском:
```bash
# 1. Закрити IDE
pkill -f Windsurf
pkill -f "Visual Studio Code"

# 2. Налаштувати .env (опціонально)
cp .env.example .env
# Відредагувати SUDO_PASSWORD
```

### Після запуску:
```bash
# 1. Дочекатися завершення (5-10 хвилин)
# 2. ПЕРЕЗАВАНТАЖИТИ MAC (обов'язково!)
sudo reboot

# 3. Встановити IDE заново
# Windsurf: https://codeium.com/windsurf
# VS Code: https://code.visualstudio.com

# 4. Запустити як новий користувач
```
